== Description ==

Puts Facebook Like and Share,Twitter,Google +1,Google buzz buttons of your choice above or below your posts.
Easy customization of active buttons and position in the Settings menu.

More info: http://wp-plugins-themes.com/wp-plugin/facebook-like-and-sharetwittergoogle-1google-buzz-buttons/


== Installation ==
1. Upload the `facebook-twitter-google` directory into the `/wp-content/plugins/` directory
2. Set your favourite values in the `Settings->facebook twittergoogle button` menu in Wordpress
3. Activate the plugin through the `Plugins` menu in WordPress
4. Enjoy!
